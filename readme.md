## 1. 프로젝트 간단 요약
간단한 미니 페이스북 서비스를 만들어 배포하는 것을 목표로 프로젝트를 진행한다.

## 2. 프로젝트 정보/개요 (진행 기간 / 목표 / 팀원)
* 진행 기간 : 23/07/12 ~ 23/8/12
* 팀장 : 박용희
* 팀원 : 김다빈, 이승연, 김소정

## 3. 프로젝트 설명 (구현한 기능 및 사진 등)
* 로그인 기능
* 피드 기능
* 연락처/친구 기능
* 개인 계정 기능

## 4. 기술 스택 (사용한 기술 스택 등)
* HTML
* CSS
* JAVASCRIPT
* REACT
